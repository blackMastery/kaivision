## üÜò Troubleshooting

**"Animations aren't working"**
‚Üí Make sure framer-motion is installed: `npm install framer-motion`

**"Styles look broken"**
‚Üí Check that globals.css is imported in your root layout
‚Üí Verify Tailwind config is correct

**"Form doesn't submit"**
‚Üí Check browser console for errors
‚Üí Ensure you updated the handleSubmit function with your API

**"Page loads slow"**
‚Üí Run `npm run build` and test production build
‚Üí Optimize images if you added any
‚Üí Check network tab in DevTools

**"Not mobile responsive"**
‚Üí This shouldn't happen - the design is mobile-first
‚Üí Check if you modified any Tailwind classes
‚Üí Test in real devices, not just DevTools
